# ConceptMap

An api designed to make the creation of web-based interactive concept maps simple and intuitive.

### Means of creation
- D3.js
- From scratch

### Ideas
- Nodes represent concepts
- Edges represent connections
- Force-directed
- Clicking can expand nodes/edges
